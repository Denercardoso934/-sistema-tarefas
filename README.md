# Sistema de Tarefas em Python

Sistema de gerenciamento de tarefas desenvolvido em Python com banco de dados SQLite.

## 📋 Funcionalidades

- Adicionar tarefas
- Listar tarefas com status
- Marcar tarefa como concluída
- Deletar tarefa

## 🚀 Como usar

1. Clone o repositório
2. Execute o arquivo `Sistema tarefa.py`
3. Escolha uma opção do menu

## 🛠️ Tecnologias

- Python 3.12
- SQLite3
